# mongat-contrib

A collection gat lisp scripts to use within [mongat](https://github.com/tsurai/mongat).

## Scripts

* [daily](https://github.com/tsurai/mongat-contrib/blob/main/daily.lisp) - Daily note creation and archiving
* [fsrs](https://github.com/tsurai/mongat-contrib/blob/main/fsrs.lisp) - Implementation of the FSRS-6 flashcard scheduling algorithm
